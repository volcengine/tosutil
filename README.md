# Volcengine Command-line Tool tosutil

tosutil is a command-line tool for accessing and managing TOS(Torch Object Storage) resource in the volcengine. This command-line tool allows you to perform batch data processing between local and TOS, addressing common business scenarios such as migrating small and medium-sized local data to the cloud, downloading small and medium-sized data from the cloud, and integrating automated script tasks. More details can be found in [document](https://www.volcengine.com/docs/6349/148775?lang=en)
